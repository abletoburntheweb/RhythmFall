# RhythmFall

Преобразует вашу музыку в уровни ритм‑игры. Полностью локальная автоматическая генерация нот — без ручного маппинга.

Примечание: Этот репозиторий содержит клиентскую часть (Godot). Репозиторий сервера (анализ аудио и генерация нот): https://github.com/abletoburntheweb/RhythmFallServer

Языки: [English](./README.md) | Русский

## Что это
RhythmFall — ритм‑игра на Godot, которая анализирует выбранный трек и на лету строит играбельную разметку нот. Лёгкий локальный сервер выполняет аудио‑анализ и возвращает ноты в игру.

## Как это работает
- Клиент (Godot) отправляет выбранную песню на локальный Python‑сервер.
- Сервер оценивает темп и ударные события, применяет жанровые паттерны и формирует ноты.
- Клиент сохраняет результат локально — можно сразу играть.

## Ключевые возможности

**Геймплей**
- Автогенерация нот из аудио (без ручной разметки)
- Паттерны для ударных: minimal, basic, enhanced, natural и custom
- 3–5 дорожек, учёт жанра, опциональное разделение на стемы
- Оценки SS–C, комбо, точность, награды XP и валюта
- Синхронизация с аудио; калибровка в настройках

**Библиотека песен**
- Свои треки + встроенные bundled songs
- Анализ BPM и генерация нот через локальный сервер
- Редактирование метаданных, история результатов по треку
- Удаление пользовательских треков из библиотеки (bundled не удаляются)

**Прогресс и мета**
- Уровень игрока и XP; внутриигровая валюта (алмазы)
- **Магазин** — обложки, темы, косметика; награды за уровни, достижения и ежедневки
- **Достижения** — цели по геймплею, библиотеке и статистике
- **Ежедневные задания** на главном меню — короткие цели с наградой валютой
- **Профиль** — любимый трек, дашборд статистики, тайлы грейдов, график точности, недавние забеги

**Меню и настройки**
- Главное меню: игра, магазин, достижения, профиль, настройки, справка
- **Настройки** — звук, графика, управление, прочее (адрес/порт сервера генерации, сброс данных)
- **Справка** — гайд с блоком «С чего начать» (тексты правятся через JSON)

**Технически**
- Локальный анализ и генерация (localhost или сервер в LAN)
- Установщик Windows и portable-сборка с v1.1.10

## Быстрый старт
- Запустите локальный сервер ([RhythmFallServer](https://github.com/abletoburntheweb/RhythmFallServer)).
- Откройте клиент Godot и запустите игру.
- Сгенерируйте ноты: выберите инструмент «drums», режим (basic/enhanced), число дорожек и песню.
- Играйте получившийся уровень.

## Скачивание для Windows

Готовые сборки для Windows доступны с **v1.1.10** (установщик и опциональный portable ZIP) — редактор Godot не нужен.

Актуальный **`RhythmFall-*-setup.exe`** — в [GitHub Releases](https://github.com/abletoburntheweb/RhythmFall/releases) (сейчас **v1.1.11**). Для той же версии может быть и portable ZIP (папка с `RhythmFall.exe`).

**Установка:** запустить setup.exe → выбрать папку (по умолчанию Program Files или «Локальные программы») → ярлык в меню Пуск, опционально на рабочий стол.

**Удаление после установщика:** Параметры → Приложения → RhythmFall → **Удалить** (или пункт «Uninstall RhythmFall» в меню Пуск). Система запустит деинсталлятор установщика — он снесёт файлы игры из папки установки и ярлыки. Отдельно спросит, удалять ли сохранения в `%APPDATA%\RhythmFall\` (по умолчанию — нет). Отдельный `Uninstall-RhythmFall.bat` в setup не входит — он только для portable ZIP.

**Сохранения и настройки** лежат не в папке игры, а в `%APPDATA%\RhythmFall\` (прогресс, ноты, покупки в магазине и т.д.). Обновление поверх старой версии через setup.exe сохранения не трогает.

**Генерация нот по-прежнему требует локальный Python‑сервер** — в скачиваемый клиент входит только игра.

## Примечания
- Репозиторий не содержит аудио; музыка и временные данные остаются локально.
- Генерация и анализ полностью выполняются на вашем устройстве, без отправки треков в сеть.
- Качество генерации зависит от микса и жанра; режим enhanced точнее, но медленнее.
- Стемы ускоряют/улучшают детекцию ударных, но заметно увеличивают время обработки — отключайте для быстрых тестов.
- Поддерживаются распространённые форматы аудио; для нестандартных кодеков возможны ограничения.
- Для расширенной конфигурации и описания пайплайна используйте репозиторий сервера.
